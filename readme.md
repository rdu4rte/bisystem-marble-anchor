## Bisys Marble-Anchor

### Instalation:
~~~
python3 -m venv <venv-folder-name>
~~~

~~~
virtualenv <venv-folder-name>
~~~

~~~
source <venv-folder-name>/bin/activate
~~~

### Install requirements
~~~
task install
~~~

### Update requirements
~~~
task freeze
~~~

### Format files
~~~
task format
~~~

### Run seed and up db containers
~~~
task run
~~~

### Stop services
~~~
task stop
~~~

## Restart services
~~~
task restart
~~~
